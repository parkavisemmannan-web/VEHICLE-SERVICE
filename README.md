# 🚗 Vehicle Service Management System

## 📖 Overview
The Vehicle Service Management System is a Java-based application developed to simplify the management of vehicle service center operations. It helps maintain customer details, vehicle information, service records, and billing in an organized and efficient manner.

## ✨ Features
- Add and manage customer details
- Store vehicle information
- Create and manage service records
- Track vehicle service status
- Generate service bills
- View service history
